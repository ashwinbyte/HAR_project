# Data

This project uses the WISDM v1.1 dataset for training and evaluation.

## Download
Please download the raw dataset from the official WISDM website:
https://www.cis.fordham.edu/wisdm/dataset.php


data/WISDM_ar_v1.1_raw.txt

## Notes
- Processed training and test files are not included in this public repository.
- They can be regenerated by running the preprocessing script.
- Personal phone recordings are not included for privacy reasons.
